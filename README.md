### 💡 주요 기능
- **배경 음악**
  - 배경 음악을 삽입하여 몰입감을 제공
  
- **충돌 처리**
  - 충돌 시 일정 시간 무적 상태가 되어 생존 유지
  
- **아이템 획득**
  - 무기를 업그레이드 하여 게임의 재미 상승
